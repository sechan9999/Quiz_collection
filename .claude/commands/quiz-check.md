# Quiz Check - 정답 정확성 검증

모든 퀴즈 문제의 정답이 실제로 정확한지 검증합니다.

## 입력 파라미터

- **카테고리** (선택): $ARGUMENTS
- 카테고리 지정 시 해당 카테고리만 검증
- 미지정 시 전체 문제 검증

## 수행 작업

1. `quiz-master/questions.js`와 `datasci-trivia/questions.js` 파일을 읽습니다.
2. 각 문제에 대해 다음을 검증합니다:
   - 정답 인덱스가 유효한 범위(0-3)인지
   - 정답 내용이 실제로 맞는지 (사실 확인)
   - 해설(explanation)이 정답과 일치하는지
3. 의심스러운 문제를 보고합니다.

## 검증 항목

### 1. 구조적 검증
| 항목 | 검증 내용 |
|-----|----------|
| 정답 인덱스 | 0-3 범위 내 정수인지 |
| 선택지 개수 | 정확히 4개인지 |
| 필수 필드 | question, options, correct/answer 존재 여부 |

### 2. 내용 검증
| 카테고리 | 검증 방법 |
|---------|----------|
| general | 일반 상식 사실 확인 |
| science | 과학적 사실/공식 검증 |
| history | 역사적 사실/연도 확인 |
| geography | 지리 정보 정확성 확인 |
| ml | 머신러닝 개념 정확성 |
| stats | 통계학 이론 검증 |
| python | Python 문법/함수 확인 |
| data | 데이터분석 개념 검증 |

### 3. 일관성 검증
- 정답과 해설의 일치 여부
- 문제와 정답의 논리적 연결
- 중복 문제 존재 여부

## 출력 형식

```

**검증 범위**: [전체 / 특정 카테고리]
**총 문제 수**: N개
**검증 완료**: N개

### 구조적 오류
| 번호 | 카테고리 | 문제 | 오류 내용 |
|-----|---------|------|----------|
| ... | ... | ... | ... |

### 정답 의심 문제
| 번호 | 카테고리 | 문제 | 현재 정답 | 의심 사유 |
|-----|---------|------|----------|----------|
| ... | ... | ... | ... | ... |

### 해설 불일치
| 번호 | 카테고리 | 정답 | 해설 | 불일치 내용 |
|-----|---------|-----|------|-----------|
| ... | ... | ... | ... | ... |

### 중복 문제
| 문제 | 위치1 | 위치2 |
|-----|------|------|
| ... | ... | ... |

### 검증 요약
- 구조적 오류: N개
- 정답 의심: N개
- 해설 불일치: N개
- 중복 문제: N개
- **검증 통과**: N개 / 총 N개
```

## 사용 예시

```
/project:quiz-check              # 전체 문제 검증
/project:quiz-check science      # science 카테고리만 검증
/project:quiz-check ml           # ml 카테고리만 검증
```

## 주의사항

- 시간에 따라 변할 수 있는 정답(인구, 순위 등)은 별도 표시
- 검증 결과 오류 발견 시 수동 확인 필요
- 자동 수정하지 않고 보고만 수행
