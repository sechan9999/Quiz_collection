# Teacher Dashboard - 선생님 모드 통합 대시보드

선생님 모드의 모든 기능을 하나의 통합 인터페이스에서 실행합니다.

## 입력 파라미터

- **실행 모드**: $ARGUMENTS
- `full` - 전체 대시보드 실행 (기본값)
- `quick` - 빠른 요약만
- `daily` - 일일 점검
- `weekly` - 주간 리포트
- `menu` - 대화형 메뉴

## 하위 명령어 연동

| 대시보드 기능 | 연동 명령어 |
|-------------|-----------|
| 학생 관리 | `/project:teacher-students` |
| 성적 조회 | `/project:teacher-scores` |
| 비교 분석 | `/project:teacher-compare` |
| 심층 분석 | `/project:teacher-analytics` |
| 리포트 생성 | `/project:teacher-report` |

## 사용 예시

```
/project:teacher-dashboard              # 전체 대시보드
/project:teacher-dashboard full         # 전체 대시보드
/project:teacher-dashboard quick        # 빠른 요약
/project:teacher-dashboard daily        # 일일 점검
/project:teacher-dashboard weekly       # 주간 리포트
/project:teacher-dashboard menu         # 대화형 메뉴
```

## 알림 설정

### 자동 알림 조건
- 학생 성적 20% 이상 급락
- 3회 연속 성적 하락
- 7일 이상 미응시
- 반 평균 10% 이상 하락

### 알림 형식
```
⚠️ [알림] 2024-01-31 14:30
━━━━━━━━━━━━━━━━━━━━━━━━
유형: 성적 급락
학생: 홍길동 (STU001)
내용: 최근 점수 850→650 (23% 하락)
권장: 개별 면담 필요
```

## 단축키 (Menu 모드)

| 키 | 기능 |
|---|------|
| S | 학생 관리 |
| C | 비교 분석 |
| A | 심층 분석 |
| R | 리포트 생성 |
| E | 성적 입력 |
| Q | 종료 |

## 권장 사용 패턴

| 시점 | 권장 모드 | 목적 |
|-----|---------|------|
| 매일 아침 | `daily` | 일일 점검 |
| 수업 중 | `quick` | 빠른 현황 파악 |
| 주말 | `weekly` | 주간 리뷰 |
| 상담 시 | `full` | 상세 분석 |
| 월말 | `full` + `report` | 성적표 발급 |
