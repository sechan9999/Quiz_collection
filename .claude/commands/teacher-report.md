# Teacher Report - 성적표/리포트 생성

학생 개인별 또는 반별 성적표와 분석 리포트를 생성합니다.

## 입력 파라미터

- **리포트 유형**: $1 (첫 번째 인자)
- **대상**: $2 (학생ID, 반, 또는 'all')
- **옵션**: $3 (기간, 형식 등)
- 입력 형식: `$ARGUMENTS` = "유형 대상 [옵션]"

## 사용 가능한 리포트 유형

| 유형 | 설명 | 형식 |
|-----|------|-----|
| `student` | 개인 성적표 | `student 학생ID [기간]` |
| `class` | 반별 성적표 | `class 반명 [기간]` |
| `summary` | 요약 리포트 | `summary [기간]` |
| `progress` | 성장 리포트 | `progress 학생ID` |
| `parent` | 학부모용 리포트 | `parent 학생ID` |
| `export` | 내보내기 | `export [형식] [대상]` |

## 사용 예시

```
/project:teacher-report student STU001           # 개인 성적표
/project:teacher-report student STU001 1month    # 1개월 성적표
/project:teacher-report class 1반                # 반별 성적표
/project:teacher-report summary                  # 전체 요약
/project:teacher-report progress STU001          # 성장 리포트
/project:teacher-report parent STU001            # 학부모용
/project:teacher-report export pdf all           # 전체 PDF
/project:teacher-report export excel 1반         # 1반 Excel
```

## 주의사항

- 학부모용 리포트는 민감한 비교 정보 제외
- 개인정보 보호를 위해 내보내기 파일 관리 주의
- 정기 리포트는 월말 자동 생성 권장
