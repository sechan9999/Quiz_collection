# Quiz Validation - 모호한 표현 검사

퀴즈 문제에서 시간이 지나면 정답이 변할 수 있는 모호한 표현을 찾아 검증합니다.

## 입력 파라미터

- **카테고리**: $ARGUMENTS
- 카테고리가 지정되면 해당 카테고리의 문제만 검증
- 카테고리가 비어있으면 전체 문제를 검증

### 사용 가능한 카테고리
- **quiz-master**: general, science, history, geography
- **datasci-trivia**: ml, stats, python, data

## 검사 대상 표현

| 표현 | 모호한 이유 | 명시해야 할 기준 |
|-----|-----------|----------------|
| 가장, 제일 | 순위가 변동될 수 있음 | 측정 시점, 기준 연도 |
| 최초, 처음 | 새로운 발견으로 변경 가능 | "기록상", "공식적으로" 등 한정어 |
| 최대, 최소 | 측정 기준에 따라 다름 | 측정 단위, 기준 (면적/인구/GDP 등) |
| 최고, 최저 | 기록 갱신 가능 | 측정 시점, "현재 기준" 명시 |
| 유일 | 새로운 발견 가능 | "알려진 바로는", "현재까지" |

## 수행 작업

1. `quiz-master/questions.js`와 `datasci-trivia/questions.js` 파일을 읽습니다.
2. $ARGUMENTS 값이 있으면 해당 카테고리만, 없으면 전체 카테고리를 검사합니다.
3. 각 퀴즈 문제(question 필드)에서 모호한 표현을 검색합니다.
4. 발견된 문제마다 다음을 출력합니다:
   - 문제 내용
   - 발견된 표현
   - 권장 수정 방향 (어떤 기준을 명시해야 하는지)

## 출력 형식

```
## 검증 결과

**검증 범위**: [전체 / 특정 카테고리명]

### 모호한 표현이 발견된 문제

#### 1. [문제 내용]
- **카테고리**: ...
- **발견된 표현**: ...
- **권장 수정**: [기준 연도/측정 시점/한정어 등을 명시하도록 제안]

#### 2. ...
```

## 예시

**문제**: "세계에서 가장 큰 대양은?"
- **발견된 표현**: 가장
- **권장 수정**: 이 문제는 태평양이 정답이며 변동 가능성이 낮으므로 유지 가능. 단, "면적 기준"을 명시하면 더 명확함.

**문제**: "인구가 가장 많은 나라는?"
- **발견된 표현**: 가장
- **권장 수정**: "2024년 기준" 등 시점을 명시하거나, explanation에 "순위가 변동될 수 있음"을 추가 권장.
