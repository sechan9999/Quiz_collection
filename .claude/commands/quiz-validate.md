# Quiz Validation - 최상급 표현 검사

퀴즈 문제에서 시간이 지나면 정답이 변할 수 있는 최상급 표현을 찾아 검증합니다.

## 검사 대상 표현
- 가장, 최초, 최대, 최소, 최고, 최저
- 제일, 유일, 처음
- 세계 최고, 세계 최대, 세계 최초

## 수행 작업

1. `quiz-master/questions.js`와 `datasci-trivia/questions.js` 파일을 읽습니다.
2. 각 퀴즈 문제(question 필드)에서 위의 최상급 표현을 검색합니다.
3. 발견된 문제들을 다음 형식으로 출력합니다:
   - 파일명
   - 카테고리
   - 문제 내용
   - 발견된 표현

## 출력 형식

```
## 최상급 표현이 포함된 퀴즈 문제 목록

### [파일명]
| 카테고리 | 문제 | 발견된 표현 |
|---------|------|------------|
| ... | ... | ... |
```

## 주의사항
- 최상급 표현이 포함된 문제는 시간이 지나면 정답이 바뀔 수 있으므로 주기적인 검토가 필요합니다.
- 예: "인구가 가장 많은 나라" - 순위가 변동될 수 있음
