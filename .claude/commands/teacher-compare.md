# Teacher Compare - 학생간 성적 비교

여러 학생의 성적을 비교 분석합니다.

## 입력 파라미터

- **비교 유형**: $1 (첫 번째 인자)
- **대상**: $2, $3... (비교 대상)
- 입력 형식: `$ARGUMENTS` = "유형 대상1 대상2 ..."

## 사용 가능한 비교 유형

| 유형 | 설명 | 형식 |
|-----|------|-----|
| `students` | 학생간 비교 | `students ID1 ID2 [ID3...]` |
| `classes` | 반별 비교 | `classes 1반 2반 [3반...]` |
| `top` | 상위 N명 | `top [N] [카테고리]` |
| `bottom` | 하위 N명 | `bottom [N] [카테고리]` |
| `progress` | 성장 비교 | `progress ID1 ID2` |
| `category` | 카테고리별 비교 | `category ID1 ID2` |

## 수행 작업

### students - 학생간 직접 비교

```
📊 종합 비교

|-----|-------|-------|-------|
| 총 응시 | 15회 | 12회 | 18회 |
| 평균 점수 | 780 | 820 | 750 |
| 최고 점수 | 950 | 980 | 920 |
| 평균 정답률 | 78% | 82% | 75% |
| 평균 응답시간 | 5.2초 | 4.8초 | 6.1초 |

| 카테고리 | 홍길동 | 김철수 | 이영희 |

🏆 종합 순위: 1.김철수 > 2.홍길동 > 3.이영희
```

## 사용 예시

```
/project:teacher-compare students STU001 STU002          # 2명 비교
/project:teacher-compare students STU001 STU002 STU003   # 3명 비교
/project:teacher-compare classes 1반 2반                 # 반별 비교
/project:teacher-compare top 10                          # 상위 10명
/project:teacher-compare top 5 science                   # 과학 상위 5명
/project:teacher-compare bottom 10                       # 하위 10명
/project:teacher-compare progress STU001 STU002          # 성장 비교
/project:teacher-compare category STU001 STU002          # 카테고리별
```

## 비교 지표 설명

| 지표 | 설명 |
|-----|------|
| 평균 점수 | 전체 응시 점수의 평균 |
| 표준편차 | 점수 분포의 일관성 |
| 정답률 | 맞힌 문제 / 전체 문제 |
| 참여율 | 응시 학생 / 전체 학생 |
| 성장률 | (최근점수-초기점수)/초기점수 |

## 주의사항

- 최소 2명 이상의 학생/반을 지정해야 함
- 응시 기록이 없는 학생은 비교에서 제외
- 공정한 비교를 위해 동일 기간 데이터 사용 권장
