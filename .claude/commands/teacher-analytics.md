# Teacher Analytics - 성적 분석

학생들의 성적 데이터를 심층 분석하여 인사이트를 제공합니다.

## 입력 파라미터

- **분석 유형**: $1 (첫 번째 인자)
- **대상/옵션**: $2 (선택)
- 입력 형식: `$ARGUMENTS` = "분석유형 [대상]"

## 사용 가능한 분석 유형

| 유형 | 설명 | 형식 |
|-----|------|-----|
| `overview` | 전체 현황 분석 | `overview` |
| `distribution` | 점수 분포 분석 | `distribution [카테고리]` |
| `trend` | 시간별 추이 분석 | `trend [기간]` |
| `difficulty` | 난이도별 분석 | `difficulty` |
| `questions` | 문제별 정답률 | `questions [카테고리]` |
| `weakness` | 취약점 분석 | `weakness [학생ID/반]` |
| `prediction` | 성적 예측 | `prediction [학생ID]` |

## 사용 예시

```
/project:teacher-analytics overview              # 전체 현황
/project:teacher-analytics distribution          # 점수 분포
/project:teacher-analytics distribution science  # 과학 점수 분포
/project:teacher-analytics trend 4weeks          # 4주 추이
/project:teacher-analytics difficulty            # 난이도별 분석
/project:teacher-analytics questions general     # 일반상식 문제별
/project:teacher-analytics weakness              # 전체 취약점
/project:teacher-analytics weakness 1반          # 1반 취약점
/project:teacher-analytics prediction STU001     # 성적 예측
```

## 분석 활용 팁

1. **주간 overview** 확인으로 전체 현황 파악
2. **취약점 분석**으로 보충 학습 계획 수립
3. **문제별 분석**으로 출제 난이도 조정
4. **추이 분석**으로 학습 효과 측정
