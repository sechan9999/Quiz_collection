# Quiz Range Review - 문제 범위 검토

지정된 범위의 퀴즈 문제를 검토하고 난이도 및 정답 분포를 분석합니다.

## 입력 파라미터

- **시작 번호**: $1 (첫 번째 인자)
- **끝 번호**: $2 (두 번째 인자)
- 입력 형식: `$ARGUMENTS` = "시작번호 끝번호" (예: "1 10")

## 문제 번호 체계

### quiz-master/questions.js
| 번호 | 카테고리 |
|-----|---------|
| 1-10 | general (일반 상식) |
| 11-20 | science (과학) |
| 21-30 | history (역사) |
| 31-40 | geography (지리) |

### datasci-trivia/questions.js
| 번호 | 카테고리 | 난이도 |
|-----|---------|-------|
| 41-45 | ml | easy |
| 46-50 | ml | medium |
| 51-55 | ml | hard |
| 56-60 | stats | easy |
| 61-65 | stats | medium |
| 66-70 | stats | hard |
| 71-75 | python | easy |
| 76-80 | python | medium |
| 81-85 | python | hard |
| 86-90 | data | easy |
| 91-95 | data | medium |
| 96-100 | data | hard |

## 수행 작업

1. `quiz-master/questions.js`와 `datasci-trivia/questions.js` 파일을 읽습니다.
2. $1번부터 $2번까지의 문제를 추출합니다.
3. 다음 항목을 분석합니다:
   - **난이도 분석**: 문제 길이, 선택지 유사도, 전문 용어 사용 여부
   - **정답 분포**: 정답 위치(0,1,2,3)의 분포가 균등한지 확인

## 출력 형식

```
## 문제 범위 검토 결과

**검토 범위**: $1번 ~ $2번 (총 N문제)

### 문제 목록
| 번호 | 카테고리 | 문제 (앞 30자) | 정답 위치 |
|-----|---------|--------------|----------|
| ... | ... | ... | ... |

### 난이도 분석
- **쉬움**: N문제 (단순 사실, 짧은 문제)
- **보통**: N문제 (개념 이해 필요)
- **어려움**: N문제 (복합 개념, 전문 용어)

### 정답 분포
| 정답 위치 | 개수 | 비율 |
|----------|-----|-----|
| 0 (첫 번째) | N | N% |
| 1 (두 번째) | N | N% |
| 2 (세 번째) | N | N% |
| 3 (네 번째) | N | N% |

### 권장 사항
- 정답 분포가 불균등하면 조정 권장
- 특정 난이도에 치우쳐 있으면 균형 조정 권장
```

## 사용 예시

- `/project:quiz-range 1 10` → 1번~10번 문제 검토 (general 카테고리)
- `/project:quiz-range 31 40` → 31번~40번 문제 검토 (geography 카테고리)
- `/project:quiz-range 41 55` → 41번~55번 문제 검토 (ml 전체)
- `/project:quiz-range 1 100` → 전체 문제 검토
