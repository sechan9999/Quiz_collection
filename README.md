# 🎮 Quiz Collection

재미있고 교육적인 인터랙티브 퀴즈 게임 모음입니다.

## 🎯 포함된 퀴즈

### 🧠 Quiz Master
일반 상식, 과학, 역사, 지리 등 다양한 주제의 퀴즈로 지식을 테스트하세요!

**카테고리:**
- 📚 일반 상식
- 🔬 과학
- 🏛️ 역사
- 🌍 지리

**특징:**
- 4개 카테고리, 각 10문제
- 30초 타이머
- 시간 보너스 점수 시스템
- 문제별 해설 제공

### 🧪 DataSciTrivia
데이터 사이언스 전문 퀴즈! 머신러닝, 통계학, Python, 데이터분석 지식을 점검하세요.

**카테고리:**
- 🤖 머신러닝
- 📊 통계학
- 🐍 Python
- 📈 데이터분석

**특징:**
- 3단계 난이도 (쉬움/보통/어려움)
- 카테고리별 15문제
- 15초 타이머
- 실시간 점수 및 통계

## 🚀 실행 방법

### 로컬에서 실행
1. 저장소를 클론합니다:
   ```bash
   git clone https://github.com/sechan9999/quiz-collection.git
   ```
2. `index.html` 파일을 브라우저에서 엽니다.

### GitHub Pages
[https://sechan9999.github.io/quiz-collection](https://sechan9999.github.io/quiz-collection)
